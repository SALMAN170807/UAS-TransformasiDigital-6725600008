Deskripsi Proyek
-----------------------------------------------------------------------------------------------------------------------------
Proyek ini merupakan studi analisis yang membandingkan penggunaan server fisik (on-premise) dengan cloud free tier sebagai media penyimpanan data menggunakan lingkungan virtual.

Simulasi dilakukan menggunakan VirtualBox dan Ubuntu Server untuk merepresentasikan server lokal, sedangkan Google Drive Free Tier digunakan sebagai alternatif penyimpanan cloud.

Selain membandingkan biaya operasional, penelitian ini juga mengevaluasi penggunaan sumber daya server seperti CPU, RAM, dan jaringan menggunakan aplikasi monitoring Netdata.

Tujuan Proyek
-----------------------------------------------------------------------------------------------------------------------------
Penelitian ini bertujuan untuk:

- Membangun simulasi server lokal menggunakan VirtualBox dan Ubuntu Server.
  
- Mengimplementasikan penyimpanan berbasis Cloud Free Tier menggunakan Google Drive.
  
- Menganalisis penggunaan sumber daya server.
  
- Membandingkan biaya operasional server lokal dengan layanan cloud gratis.
  
- Memberikan rekomendasi penggunaan infrastruktur berdasarkan efisiensi biaya dan kebutuhan pengguna.
